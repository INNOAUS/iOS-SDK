This Library support iPhone 4S over and iOS 7.1 over.

== Known Issue ==

1. Device iPhone 4S has not support configuration schema "Debug"
If you build for device iPhone 4S, you must set schema to "Release".
If you build for device over iPhone 5 or 5S, this library support "Debug" and "Release" all.
